# Thesis terminology correction

Use **Multi-Parameter Wi-Fi Detection Framework** in place of **Multi-Sensor
Monitoring Architecture** throughout the thesis, diagrams, presentation, and
paper. SentinelShield combines multiple Wi-Fi observations—RSSI, BSSID, SSID,
channel, encryption, AP density, temporal signal variation, fingerprint
similarity, reputation, rule-based indicators, and ML outputs—but it does not
currently collect data from multiple independent physical sensors.

Suggested thesis sentence: “SentinelShield implements a multi-parameter Wi-Fi
detection framework that fuses signal, configuration, identity, historical,
and machine-learning indicators to identify rogue access points.”
